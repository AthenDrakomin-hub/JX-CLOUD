# JX Cloud 管理员手册

## 系统概述

JX Cloud 是一款现代化的企业级酒店管理套件，采用 React 19 与 Supabase 云原生架构，为酒店、餐厅及度假村提供全链路管理解决方案。

## 管理员权限

作为管理员，您拥有以下权限：
- 所有功能模块的完全访问权限
- 用户账户管理
- 系统配置
- 安全审计日志查看

## 用户管理

### 创建新用户
1. 进入"员工管理"模块
2. 点击"添加用户"
3. 填写用户信息（用户名、姓名、角色）
4. 角色包括：管理员(admin)、经理(manager)、员工(staff)

### 删除用户
1. 在用户列表中找到目标用户
2. 点击删除按钮
3. 确认删除操作

## 系统配置

### 环境变量配置
- VITE_SUPABASE_URL: Supabase项目URL
- VITE_SUPABASE_ANON_KEY: Supabase匿名密钥

### 实时订阅设置
确保在Supabase Dashboard的Database -> Replication中为orders表启用Realtime选项

## 安全管理

### 行级安全 (RLS)
- 运行database/enable_rls.sql脚本启用安全策略
- 审计日志自动记录所有敏感操作

### 安全审计
- 查看安全审计日志监控系统活动
- 定期检查登录记录

## 系统维护

### 数据备份
- 定期导出数据库备份
- 检查存储桶配置

### 性能监控
- 监控订单处理速度
- 检查系统响应时间
- 查看实时通知状态

## 问题排查

### 常见问题
1. 登录失败：检查用户名密码是否正确
2. 数据同步异常：确认Supabase连接状态
3. 实时通知失效：检查Realtime订阅设置
