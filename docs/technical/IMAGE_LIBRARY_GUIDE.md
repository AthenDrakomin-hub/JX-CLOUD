# 图片素材库使用指南

## 概述
JX Cloud的图片素材库允许管理员上传和管理图片资源，这些图片可以用于菜单项、房间展示等场景。上传的图片对所有用户公开可见。

## 上传图片

1. 导航到"素材库"标签页
2. 点击上传按钮（云上传图标）
3. 输入素材名称
4. 选择分类（主菜、海鲜、主食、汤类、饮料、甜品等）
5. 选择本地图片文件（支持PNG, JPEG, JPG, WEBP, GIF格式，最大5MB）
6. 点击上传

## 在菜单管理中使用图片

1. 导航到"菜单管理"标签页
2. 添加或编辑菜品
3. 在图片URL字段中，可以从素材库中复制图片链接
4. 素材库中的图片可以直接用于菜单项，所有用户都能看到

## 图片管理

- 上传者可以删除自己的图片
- 图片URL可以复制并用于其他模块
- 支持按分类筛选图片
- 支持按名称搜索图片

## 安全策略

- 只有认证用户可以上传、更新和删除图片
- 所有用户都可以查看图片（用于菜单展示等共享用途）
- 文件大小限制为5MB以防止恶意上传
- 文件名自动生成以避免冲突
