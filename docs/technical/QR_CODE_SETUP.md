# 房间二维码点餐配置指南

## 功能说明

房间二维码点餐功能允许客人通过扫描房间对应的二维码直接进入点餐页面，无需登录系统或进行任何身份验证。

## 配置说明

### 1. 二维码生成

- 在房间管理页面中，每个房间都显示一个二维码
- 二维码包含房间号信息，格式为：`https://yourdomain.com?room=房间号`
- 例如：`https://www.jiangxijiudian.store?room=8201`

### 2. 点餐流程

1. 客人扫描房间二维码
2. 自动进入点餐页面，房间号已预设
3. 客人可以浏览菜单并下单
4. 订单将关联到对应的房间号

### 3. 技术实现

- 二维码使用 `qrcode.react` 库生成
- 通过 URL 参数 `?room=房间号` 识别房间
- GuestOrder 组件处理房间参数并显示对应房间的点餐界面
- 无需 Supabase 身份验证，客人可直接点餐

### 4. 批量打印

房间管理页面提供批量打印功能，可以一次性为多个房间生成二维码。

### 5. 安全考虑

- 二维码只包含房间信息，不包含敏感数据
- 订单提交时会关联到对应的房间号
- 确保房间号的唯一性

## 使用说明

1. 在房间管理页面生成房间二维码
2. 将二维码放置在对应房间/桌位
3. 客人扫描二维码进入点餐页面
4. 客人完成点餐后，订单将自动关联到房间号

## 自定义配置

如需自定义二维码样式或点餐页面，请修改以下文件：

- `components/RoomGrid.tsx` - 房间管理页面，包含二维码生成功能
- `components/GuestOrder.tsx` - 客人点餐页面，处理房间参数
- `App.tsx` - 主应用，处理路由和房间参数解析
