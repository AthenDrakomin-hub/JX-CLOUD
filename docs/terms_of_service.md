# 系统服务条款与使用协议

## 1. 服务范围

### 1.1 服务内容
JX Cloud酒店管理系统（以下简称"本系统"）为酒店、餐厅及度假村提供以下服务：
- 房间/桌位管理
- 在线订单处理
- 菜单配置管理
- 财务审计功能
- 员工权限管理
- 多语言支持服务

### 1.2 服务限制
- 本系统为辅助管理工具，不替代人工决策
- 系统运行依赖网络连接，网络异常可能影响服务
- 财务审计功能仅供参考，最终以实际财务数据为准

## 2. 责任界定

### 2.1 开发商责任
- 提供系统正常运行保障
- 确保数据传输安全加密
- 提供必要的技术支持
- 定期进行系统安全更新

### 2.2 用户责任
- 妥善保管账号密码
- 按照操作手册使用系统
- 不得进行逆向工程或破解系统
- 遵守当地法律法规使用系统

## 3. 数据安全与隐私保护条款

### 3.1 零PII收集原则
本系统在访客（住客）端点餐过程中，**不收集**任何个人识别信息（PII），包括但不限于：姓名、手机号、证件号、地理位置坐标或社交媒体账户。系统仅记录房间 ID 以实现精准送餐。

### 3.2 数据持久化策略
- 访客的浏览偏好仅在点餐期间作为临时缓存处理
- 订单完成后24小时内进行数字粉碎或自动脱敏
- 确保访客的行为隐私

### 3.3 员工权限最小化
- 系统管理端实行严格的角色访问控制（RBAC）
- 普通前厅员工仅能查看与其职责相关的点餐状态
- 无法访问后台数据库配置或财务支出详情

### 3.4 数据加密
- 所有数据传输采用SSL/TLS加密
- 用户密码采用哈希加密存储
- 敏感信息通过环境变量隔离

## 4. 服务级别协议(SLA)

### 4.1 系统可用性
- 承诺系统可用性：99.5%
- 计划维护将提前48小时通知
- 紧急维护将尽快完成并通知用户

### 4.2 性能指标
- 页面响应时间：平均<3秒
- 数据同步延迟：<5秒
- API响应时间：<2秒

### 4.3 技术支持
- 工作时间：周一至周五 9:00-18:00
- 紧急支持：24小时内响应
- 一般咨询：48小时内响应
