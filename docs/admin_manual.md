# 系统管理员手册

## 概述

本手册适用于系统管理员，包含用户权限管理、系统配置、数据维护等高级功能。

## 用户权限管理

### 添加新用户
1. 进入"安全中心"模块
2. 点击"员工名录"
3. 点击"注册新员工"按钮
4. 填写员工信息
5. 选择对应权限
6. 点击"保存"

### 权限配置说明
- **菜单编辑权限**: 可编辑菜单项、价格和库存
- **财务记录权限**: 可查看财务流水
- **订单处理权限**: 可处理订单出餐
- **安全中心权限**: 可访问安全中心
- **系统配置权限**: 可修改核心系统设置
- **资产维护权限**: 可管理云端素材

### 用户状态管理
- **激活状态**: 正常登录使用系统
- **封禁状态**: 禁止登录系统
- 修改用户状态: 在员工名录中编辑对应用户

## 系统设置

### 基础设置
1. 进入"系统设置"模块
2. 修改机构名称（显示在账单和收据上）
3. 保存设置

### 数据管理
- **重置系统**: 清除所有本地缓存数据并恢复初始状态
- 注意：此操作不可逆，请谨慎操作

## 数据备份与恢复

### 数据备份
系统自动进行数据备份，备份状态可在系统设置中查看

### 数据恢复
如需数据恢复，请联系技术支持团队

## 安全管理

### 审计日志
1. 进入"安全中心"
2. 点击"审计日志"
3. 查看所有操作记录
4. 根据时间、用户或操作类型筛选日志

### 安全策略
- 定期更换管理员密码
- 监控异常登录行为
- 定期检查用户权限分配

## 常见管理任务

### 1. 房间管理
- 系统默认创建64个房间（8201-8232和8301-8332）
- VIP房间包括：vip666, vip888, vip000

### 2. 菜单管理
- 添加新菜品
- 设置菜品分类
- 管理菜品库存
- 设置推荐菜品

### 3. 财务管理
- 添加运营支出
- 查看收入明细
- 生成财务报表

## 系统维护

### 日常维护任务
- 检查系统运行状态
- 监控数据同步情况
- 审查用户权限设置
- 备份重要数据

### 性能优化建议
- 定期清理不必要的数据
- 优化数据库查询
- 监控系统响应延迟
