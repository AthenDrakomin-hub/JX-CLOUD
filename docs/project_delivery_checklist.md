## 技术规格

### 1. 前端技术栈
- React 19
- TypeScript
- Vite 6.x
- Lucide React 图标
- Recharts 数据可视化

### 2. 后端技术栈
- Supabase (PostgreSQL)
- Supabase Realtime
- JWT 认证
- 行级安全 (RLS)

### 3. 性能指标
- 页面加载时间 < 3秒
- API响应时间 < 2秒
- 系统可用性 ≥ 99.5%
- 数据同步延迟 < 5秒

### 4. 安全标准
- SSL/TLS加密传输
- 行级安全策略
- 角色基础访问控制
- 安全审计日志

## 部署要求

### 1. 环境要求
- Node.js >= 18.0.0
- npm >= 8.0.0
- Supabase项目配置

### 2. 配置要求
- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY
- GEMINI_API_KEY (可选)

### 3. 服务器要求
- 静态文件托管
- HTTPS支持
- CDN配置 (推荐)
