# 用户操作手册

## 概述

本手册旨在帮助酒店工作人员和管理人员快速掌握JX Cloud酒店管理系统的各项功能操作。

## 登录系统

1. 打开系统网址
2. 输入用户名和密码
3. 点击"登录"按钮进入系统

## 功能模块操作指南

### 1. 仪表盘（经营分析）
- 查看实时经营数据
- 监控收入、支出、入住率等关键指标
- 分析各品类收入情况

### 2. 房间/桌位管理
- 查看所有房间/桌位状态
- 绿色表示空闲，红色表示正在使用
- 点击房间可查看详细订单信息

### 3. 订单处理（后厨出餐）
- 查看待处理订单
- 点击"开始制作"处理订单
- 点击"配送"将订单发送到配送环节
- 点击"确认送达"完成订单

### 4. 菜单配置
- 添加新菜品
- 编辑现有菜品信息
- 设置菜品价格和库存
- 管理菜品分类

### 5. 财务审计
- 查看收入明细
- 管理支出记录
- 生成财务报表

### 6. 支付管理
- 配置支付渠道
- 管理支付方式
- 查看支付状态

### 7. 安全中心
- 管理员工账户
- 设置权限
- 查看安全日志

### 8. 系统设置
- 修改系统参数
- 重置系统数据
- 查看系统状态

## 常用操作技巧

1. 使用快捷键快速处理订单
2. 利用搜索功能快速查找菜品
3. 通过筛选功能查看特定订单状态
4. 使用批量操作提高工作效率

## 多语言切换

点击页面右上角的语言选择器，可切换中文、英文、塔加路语三种语言。
