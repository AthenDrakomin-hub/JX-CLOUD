# JX Cloud 用户手册

## 概述

JX Cloud（江西云厨）是专为现代化酒店、高端餐饮及综合度假村打造的全链路管理系统。本手册将指导您如何使用系统的基本功能。

## 登录系统

1. 打开系统网页：https://www.jiangxijiudian.store/
2. 输入您的用户名和密码
3. 点击"初始化会话"按钮登录

## 语言切换

系统支持中文、英文、菲律宾语三语切换：
- 点击右上角语言选择器
- 选择您需要的语言

## 主要功能模块

### 仪表板 (Dashboard)
- 查看实时业务数据
- 监控订单状态
- 查看房间状态

### 房间管理 (Rooms)
- 查看所有房间状态
- 生成房间二维码供客人点餐

### 订单管理 (Orders)
- 查看所有订单
- 更新订单状态（待处理 → 准备中 → 配送中 → 已完成/已取消）

### 菜单管理 (Menu)
- 查看和管理菜单
- 更新菜品信息
- 管理库存

### 财务管理 (Finance)
- 查看收入支出
- 添加支出记录

### 员工管理 (Users)
- 管理员工账户
- 分配权限

## 安全注意事项

- 请妥善保管您的登录凭据
- 定期更换密码
- 登出时请确保点击退出按钮
