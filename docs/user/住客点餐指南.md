# 住客点餐指南

## 欢迎使用JX Cloud点餐服务

亲爱的住客，欢迎使用JX Cloud智能点餐服务！本指南将帮助您轻松完成房间点餐。

## 开始点餐

### 方式一：扫描二维码
1. 在您的房间内找到专属二维码
2. 使用手机相机或微信扫描二维码
3. 系统将自动识别您的房间号并进入点餐界面

### 方式二：手动输入房间号
1. 访问点餐网址：https://www.jiangxijiudian.store/
2. 在页面提示处输入您的房间号（例如：8201）
3. 点击确认进入点餐界面

## 浏览菜单

### 查看菜品
- 菜单按类别分类（主菜、海鲜、主食、汤品、饮品、甜品）
- 每道菜品显示名称、价格、图片和库存情况
- 点击菜品可查看详细信息

### 语言切换
- 点击右上角语言选择器
- 可选择中文、英文或菲律宾语
- 菜单将自动切换到您选择的语言

## 点餐流程

### 选择菜品
1. 浏览菜单，选择您喜欢的菜品
2. 点击菜品旁的"+"按钮添加到购物车
3. 设置所需数量
4. 重复以上步骤选择更多菜品

### 确认订单
1. 点击右下角购物车图标查看订单
2. 核对所选菜品和数量
3. 确认房间号无误
4. 点击"提交订单"按钮

### 支付方式
系统支持多种支付方式：
- GCash
- Maya
- GrabPay
- 信用卡/借记卡
- 房账（挂房费）
- 现金

## 订单跟踪

### 查看订单状态
- 提交订单后可在订单详情页查看状态
- 状态包括：待处理 → 准备中 → 配送中 → 已完成
- 系统会实时更新订单状态

### 联系服务
- 如需特殊要求，可在订单备注中说明
- 如有疑问，请联系前台服务

## 服务特色

### 快速配送
- 订单确认后将立即处理
- 预计配送时间将在订单中显示

### 24小时服务
- 提供全天候点餐服务
- 深夜也可享受美味餐点

### 多语言支持
- 支持中文、英文、菲律宾语
- 确保沟通无障碍

## 常见问题

### Q: 扫码后无法进入点餐界面？
A: 请确认房间号是否正确，或联系前台获取帮助。

### Q: 菜品显示无库存怎么办？
A: 请稍后重试或选择其他菜品，库存会实时更新。

### Q: 如何修改已提交的订单？
A: 如订单状态仍为"待处理"，可联系前台协助修改。
